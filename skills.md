# skills/marketplace-engine.md

## Description
Core tech directives for building the custom HC Group full-stack marketplace from scratch.

## Technical Scope
- **Database Architecture:** Relational system optimized for highly complex variant matrices (e.g., lighting fixtures by color temperature, furniture by dimension, bulk pricing rules for raw materials).
- **Security & Logistics:** Custom local state/city shipping matrix calculations to directly target trust factors against shipping delays.
- **UI Guidelines:** Clean, generous whitespace, sharp typography, dark luxury blue and gold color scheme matching the logo design assets.
